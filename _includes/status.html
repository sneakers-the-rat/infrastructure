Contributing

This is a living document, and feedback is welcome!

Annotation

This page embeds hypothes.is, which allows you to annotate the text! Existing annotations should show up highlighted like this, and you can make a new annotation by highlighting an area of the text and clicking the 'annotate' button that appears! You can see all annotations by opening the sidebar on the right.

Discussion

The document repository has an associated gitter chat for public discussion.

Revisions

If you would like to propose additions or revisions, please submit a pull request to the document repository. Please indicate if you would like to be added to to the authorship list, and I will rebuild the document and resubmit to arxiv.
